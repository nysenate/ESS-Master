Example View Travel Applications

Below is an example of the View Travel Applications screen as seen by all Senate employees. View Travel Applications screen allows you to list all Application for Travel Approval forms you have generated for review and printing.

Fields Explained

Refer to the chart to find basic definitions for fields.

Field Name

Definition

Travel application on behalf of

This is your name or name of the employe for which you generated a Application for Travel Approval form.

From

This specifies the begin date of the date range for which you are searching for generated Application for Travel Approval forms.

To

This specifies the end date of the date range for which you are searching for generated Application for Travel Approval forms.
