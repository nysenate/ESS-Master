Example Acknowledgments

Below is an example of the Acknowledgments screen as seen by all Senate employees. The Acknowledgments screen enable you to review policies/documents which which require your review and acknowledgment annually.

 

Fields Explained

Refer to the chart to find basic definitions for fields.

Field Name

Definition

Pending Acknowledgments

This area will display any outstanding policies/documents that you need to review and acknowledge.

Completed Acknowledgments

This area will list all policies/documents that you reviewed and the date each was acknowledged.
